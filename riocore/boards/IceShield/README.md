# IceShield
**RIO-IceShield board for Raspberry PI4**

* URL: [https://github.com/multigcs/rio-iceshield](https://github.com/multigcs/rio-iceshield)
* Toolchain: icestorm
* Family: ice40
* Type: up5k
* Package: sg48
* Flashcmd: ./spiflash.sh rio.bin
* Clock: 18.000Mhz -> PLL -> 29.812Mhz (Pin:35)
